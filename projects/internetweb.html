Internet

Definition: A global network of eletronic devices that commnunicate with each other.

Web

Definition: All the public websites that users can access through the internet.

How are the Internet and Web different?

The Internet is the network, while the Web is a service that uses the network(The Internet).

ASCII and Unicode

Definition: These are two popular encoding schemes. ASCII is for symbols, digits, letters, etc., whereas Unicode encodes special texts from different languages.

What is UTF-8

Definition: A variable-length character encoding standard used for Unicode Standard.

URL

Definition: The address of a unique resource on the Internet

Domain Name

Definition: A component of a URL. (i.e. for 'www.ChurchofJesusChrist.org', 'ChurchofJesusChrist' is the domain name.)

HTTP

Definition: A protocol for transferring data between web browsers and servers. It transfers data in plaintext. It stands for 'Hypertext Transfer Protocol'.

HTTPS

Definition: It's also a protocol for transferring data between web browsers and servers. It protects data by encrypting it.

IP Adress

Definition: It's a numberical label such as 192.0.2.1 assigned to a device connected to a computer. It uses the Internet Protocol for communication.

DNS

Definition: It turns domain names into IP adresses.

W3C Recommendations

Definition: A standard by the World Wide Web Consortium.

XML, XHTML, HTML, HTML5

They are all markup languages. XML is designed to store and transport data. XHTML is HTML reformulated as an XML application. HTML is the standard markup language for creating web pages. HTML5 is the latest version of HTML.
