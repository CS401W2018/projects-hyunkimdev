What is the difference between a block and an inline tag? Give examples of each one.

Block tags take up the full width and start on a new line like div tag, while inline tags take up only as much space as needed and stay in line like img tag.

What is normal flow in HTML?

Normal flow is the default layout where elements are displayed from top to bottom.

What does float do? Why do you need to clear the float?

Float moves an element to the left or right, and you need to clear it to avoid layout issues.

What do absolute, fixed, and relative do?

Absolute is positioned relative to the nearest parent, fixed stays in place while scrolling, and relative is positioned based on its normal location.
