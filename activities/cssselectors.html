Different Ways to Select an Item:

You can select items by tag name, ID, class, or using other CSS selectors.

Difference Between an ID and a Class:

An ID is unique to one element, while a class can be used on multiple elements.

Difference Between a <span> Tag and a <div> Tag:

<span> is used for small pieces of text, and <div> is used for larger sections of the page.

Difference Between a <div> Tag and <header>, <main>, <footer>:

<div> is a general container, while <header>, <main>, and <footer> have specific meanings for different parts of the page.

Different Ways to Select a Color:

You can choose colors by name, hex code, RGB, or HSL in CSS.
