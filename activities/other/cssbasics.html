Explain the difference between inline, embedded, and external CSS.

Inline CSS is added directly to HTML tags, embedded CSS is placed within a <style> tag in the HTML, and external CSS is written in a separate .css file linked to the HTML.

Explain why you should only use external CSS.

It ensures consistent styling across all pages, makes updates easier, improves loading speed, and keeps HTML clean.

What is a selector used for?

A selector targets specific HTML elements to apply CSS styles to them.
